## 💁🏻‍♂ 스터디 소개

&nbsp;&nbsp;**SSAFY 테크 콘서트 스터디**는 `웹 개발 분야의 기술, CS 지식, 최신 개발 트랜드 등`의 주제로 약 10분간 발표를 통해 서로 **지식을 공유**하는 스터디입니다.

`약 10분 발표` > `5분 QnA` > `다음 발표 제목 소개(다음 발표자)` > `스터디 종료 후 md 파일 커밋`
